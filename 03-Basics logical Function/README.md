### 🟢 Sheet 03: Logical Functions & Decision Making
In this module, I implemented logic into my spreadsheets, allowing for automated data categorization.

**Key Functions Mastered:**
- **IF Statements:** Created automated "Target Status" labels.
- **Nested IFs:** Developed a tiered performance ranking system.
- **AND/OR Functions:** Combined multiple criteria for complex logical tests (e.g., Bonus Eligibility).
- **IFS Function:** Used as a modern, cleaner alternative to nested IFs for multi-tier analysis.

**Example Logic:**
`=IF(Sales > 10000, "Excellent", IF(Sales > 5000, "Good", "Needs Review"))`
