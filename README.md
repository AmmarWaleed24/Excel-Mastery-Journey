# 📊 Microsoft Excel Mastery Roadmap: From Zero to Data Hero

Welcome to my learning journey! This repository is a live documentation of my progress in mastering Microsoft Excel. I am following a structured path to transition from basic data entry to advanced data analytics.

---

## 🗺️ Learning Roadmap

### Phase 1: The Foundations (Beginner) 🟢
*Focus: Understanding the interface and basic data manipulation.*
- [ ] **Excel Basics:** Interface, Cells, Rows, Columns, and Ribbons.
- [ ] **Data Entry & Formatting:** Number formats, Cell styles, and Borders.
- [ ] **Basic Arithmetic:** SUM, AVERAGE, MIN, MAX, COUNT.
- [ ] **Flash Fill & AutoFill:** Speeding up data entry.
- [ ] **Sorting & Filtering:** Organizing datasets efficiently.

### Phase 2: Intermediate Functions & Logic 🟡
*Focus: Automating calculations and organizing data.*
- [ ] **Logical Functions:** IF, AND, OR, IFS.
- [ ] **Text Functions:** CONCAT, LEFT, RIGHT, MID, TRIM.
- [ ] **Date & Time Functions:** TODAY, NETWORKDAYS, DATEDIF.
- [ ] **Lookup Functions:** VLOOKUP, HLOOKUP, and the powerful **XLOOKUP**.
- [ ] **Math & Statistical:** SUMIFS, COUNTIFS, AVERAGEIFS.

### Phase 3: Data Analysis & Summarization 🟠
*Focus: Turning raw data into meaningful reports.*
- [ ] **Pivot Tables:** Creating summaries and grouping data.
- [ ] **Pivot Charts:** Visualizing Pivot Table data.
- [ ] **Conditional Formatting:** Highlighting trends and outliers.
- [ ] **Data Validation:** Creating drop-down lists and protecting data integrity.
- [ ] **Table Object:** Benefits of using `Ctrl + T` for dynamic ranges.

### Phase 4: Advanced Visualization & Dashboards 🔴
*Focus: Creating professional-grade interactive reports.*
- [ ] **Advanced Charting:** Combo charts, Waterfall, and Sunburst.
- [ ] **Slicers & Timelines:** Making dashboards interactive.
- [ ] **Dashboard Design Principles:** Clean UI, Color theory, and User experience.
- [ ] **Introduction to Power Query:** Cleaning "dirty" data automatically.

### Phase 5: Automation & Expert Level 💎
*Focus: Scaling efficiency with programming.*
- [ ] **Power Pivot & DAX:** Handling millions of rows and complex relationships.
- [ ] **Macro Recording:** Automating repetitive tasks.
- [ ] **Introduction to VBA:** Writing basic scripts to customize Excel.
- [ ] **Excel & Python:** (Modern Excel) Using Python for advanced analytics.
