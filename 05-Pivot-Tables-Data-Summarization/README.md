# 📊 Lesson 05: Pivot Tables - Data Summarization

## 🎯 Objectives
Transforming large, unorganized datasets into meaningful summary reports and interactive dashboards with just a few clicks.

---

## 🛠 Skills Applied
- **Data Aggregation:** Instantly calculating Sum, Average, and Count for large datasets.
- **Dynamic Grouping:** Grouping sales data by Month, Quarter, or Year.
- **Calculated Fields:** Adding custom calculations inside the Pivot Table.
- **Visual Filtering (Slicers):** Adding interactive buttons to filter the whole report.

## 📊 Practical Application
In the file `05-Pivot-Tables-Practice.xlsx`, I analyzed a sales dataset of 500+ rows to find:
1. **Top Selling Products:** A summary of total revenue per item.
2. **Sales Performance:** Total sales achieved by each salesperson.
3. **Monthly Trends:** Tracking how sales fluctuated over the year.

### Key Visual Feature:
Applied **Slicers** to allow users to filter the report by "Region" or "Category" instantly.
