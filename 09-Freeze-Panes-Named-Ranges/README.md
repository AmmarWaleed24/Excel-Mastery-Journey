# ❄️ Lesson 09: Freeze Panes & Named Ranges

## 🎯 Objectives
Improving navigation in large datasets and making formulas easier to read and maintain.

---

## 🛠 Skills Applied
- **Freeze Panes:** Locking the top row or first column so they remain visible while scrolling through thousands of records.
- **Named Ranges:** Giving a descriptive name to a group of cells (e.g., naming a price column "Unit_Price") to use it in formulas like `=SUM(Unit_Price)` instead of `=SUM(B2:B500)`.
- **Name Manager:** Editing and deleting ranges to keep the workbook organized.

## 📊 Practical Application
In the file `09-Navigation-Practice.xlsx`, I:
1. **Froze the Header Row** of a large sales table.
2. **Named the Tax Rate cell** and used the name in calculations, making the formulas self-explanatory.
