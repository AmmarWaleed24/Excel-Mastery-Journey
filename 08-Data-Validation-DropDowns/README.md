# ❌ Lesson 08: Data Validation & Drop-down Menus

## 🎯 Objectives
Preventing data entry errors and ensuring data integrity by controlling what users can type into a cell.

---

## 🛠 Skills Applied
- **List Validation:** Creating drop-down menus to select from predefined options (e.g., Status: Paid, Pending, Overdue).
- **Custom Constraints:** Restricting inputs to specific date ranges or whole numbers.
- **Input Messages:** Showing a hint when a user selects a cell.
- **Error Alerts:** Customizing the pop-up message that appears when someone enters invalid data.

## 📊 Practical Application
In the file `08-Data-Validation-Practice.xlsx`, I added:
1. A **Drop-down list** for "Department" and "Job Title".
2. A **Date constraint** to ensure "Joining Date" is not in the future.
