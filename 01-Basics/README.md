# 📁 Lesson 01 & 02: Excel Foundations & Data Formatting

## 🎯 Objectives
The goal of this lesson was to master the Excel environment and learn how to present data professionally rather than just entering raw numbers.

---

## 🛠 Skills & Concepts Applied

### 1. Interface Mastery
- **The Ribbon:** Navigating through Home, Insert, and Data tabs.
- **Cell Referencing:** Understanding how `A1`, `B10`, etc., work as unique addresses for data.
- **Workbooks vs. Worksheets:** Managing multiple sheets within a single file.

### 2. Professional Data Entry
- **Number Formatting:** Converting plain numbers into **Currency ($)**, **Percentages (%)**, and **Short Dates**.
- **Cell Styles:** Applying Heading styles to make data tables readable.
- **Flash Fill (Ctrl + E):** Used this "AI-like" feature to automatically recognize patterns and fill data without formulas.

### 3. Data Organization
- **Borders & Shading:** Creating clear boundaries for datasets.
- **Alignment:** Using wrap text and merge-center for better header presentation.
