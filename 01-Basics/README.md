# Lesson 01 & 02: Excel Foundations & Data Formatting

## 🎯 Objectives
- Understand the Excel interface (Ribbon, Sheets, Cells).
- Practice efficient data entry and formatting.

## 🛠 Skills Applied
- **Cell Referencing:** Understanding how columns and rows intersect.
- **Formatting:** Applied Currency format to prices and Date format to timelines.
- **Table Styling:** Used "Format as Table" to make data readable and filterable.
- **Flash Fill:** Used to combine first and last names or extract patterns.
